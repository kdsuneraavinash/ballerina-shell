=== Ballerina Shell Help ===

Type a ballerina statement, expression or a declaration.
Following command are the in-built commands.

/exit       - Exit the terminal shell.
/imports    - Get a list of available imports.
              Other imports need to be imported with a prefix
/debug      - Toggle debug mode. You will see debug messages.
/reset      - Reset REPL state.
              All previous code snippets will be removed from REPL memory.
/about      - About the REPL.
/help       - Get this message.

Usage of shell command.
usage: ./run.sh [OPTIONS]
 -d,--debug            start with debug mode
 -e,--executor <arg>   executor to use (reeval/asm)
 -h,--help             prints this help message
